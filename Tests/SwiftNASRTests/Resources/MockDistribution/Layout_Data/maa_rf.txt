                                    
                      MISCELLANEOUS ACTIVITY AREA
                        DATA BASE RECORD LAYOUT
                             (MAA-FILE)

INFORMATION EFFECTIVE DATE: 10/12/2017

    RECORD FORMAT: FIXED
    LOGICAL RECORD LENGTH: 919


FILE STRUCTURE DESCRIPTION:
---------------------------

    THERE ARE A VARIABLE NUMBER OF FIXED-LENGTH RECORDS TO DESCRIBE EACH 
    MISCELLANEOUS ACTIVITY AREA (MAA). THE NUMBER OF RECORDS IS DETERMINED BY 
    THE AMOUNT OF TEXTUAL INFORMATION ASSOCIATED WITH EACH MAA. EACH OF THE 
    SEVEN RECORD TYPES IS IDENTIFIED BY A RECORD TYPE INDICATOR (MAA1, MAA2, 
    ..., MAA7) AND A RECORD KEY THAT IS UNIQUE TO EACH MISCELLANEOUS ACTIVITY 
    AREA. THE RECORDS ARE IN SEQUENCE, I.E., ALL TEXT RECORDS FOLLOW THEIR 
    READABLE ORDER. THE FILE IS SORTED BY STATE AND CITY.

    EACH RECORD ENDS WITH A CARRIAGE RETURN CHARACTER AND LINE FEED CHARACTER 
    (CR/LF). THIS LINE TERMINATOR IS NOT INCLUDED IN THE LOGICAL RECORD 
    LENGTH.


DESCRIPTION OF THE RECORD TYPES: --------------------------------

    ‘MAA1’ RECORD TYPE CONTAINS THE BASE MISCELLANEOUS ACTIVITY AREA DATA. 
    THERE IS ONE MAA1 RECORD FOR EACH AREA.

    ‘MAA2’ RECORD TYPE CONTAINS THE POLYGON COORDINATES. EACH MAA2 RECORD 
    CONTAINS ONE LINE OF TEXT. THERE CAN BE ANY NUMBER OF MAA2 RECORDS.

    ‘MAA3’ RECORD TYPE CONTAINS TEXTUAL INFORMATION REGARDING ANY TIMES OF 
    USE. EACH MAA3 RECORD CONTAINS ONE LINE OF TEXT. THERE CAN BE ANY NUMBER 
    OF MAA3 RECORDS.

    ‘MAA4’ RECORD TYPE CONTAINS TEXTUAL INFORMATION REGARDING ANY USER GROUP. 
    EACH MAA4 RECORD CONTAINS ONE LINE OF TEXT. THERE CAN BE ANY NUMBER OF 
    MAA4 RECORDS.

    ‘MAA5’ RECORD TYPE CONTAINS TEXTUAL INFORMATION REGARDING ANY CONTACT 
    FACILITY FREQUENCY DATA. EACH MAA5 RECORD CONTAINS ONE LINE OF TEXT. THERE 
    CAN BE ANY NUMBER OF MAA5 RECORDS.

    ‘MAA6’ RECORD TYPE CONTAINS TEXTUAL INFORMATION REGARDING ANY CHECK FOR 
    NOTAMS DATA. EACH MAA6 RECORD CONTAINS ONE LINE OF TEXT. THERE CAN BE ANY 
    NUMBER OF MAA6 RECORDS.

    ‘MAA7’ RECORD TYPE CONTAINS TEXTUAL INFORMATION REGARDING ANY REMARKS 
    ABOUT THE MAA BASE DATA. EACH MAA7 RECORD CONTAINS ONE LINE OF TEXT. THERE 
    CAN BE ANY NUMBER OF MAA7 RECORDS.


GENERAL INFORMATION:
--------------------
     1.  LEFT JUSTIFIED FIELDS HAVE TRAILING BLANKS
     2.  RIGHT JUSTIFIED FIELDS HAVE LEADING BLANKS
     3.  ELEMENT NUMBER IS FOR INTERNAL REFERENCE ONLY
         AND NOT IN RECORD
     4.  THE MAA ID IS THE UNIQUE RECORD IDENTIFIER.
     5.  MAA ID IS REPEATED ON EACH RECORD.

     6.  LATITUDE AND LONGITUDE INFORMATION IS REPRESENTED IN TWO WAYS:

         A. FORMATTED LAT/LONG VALUES AND ARE REPRESENTED AS FOLLOWS:

            LATITUDE    DD-MM-SS.SSSSH
            LONGITUDE   DDD-MM-SS.SSSSH

            WHERE :     DD IS DEGREES
                        MM IS MINUTES
                        SS.SSSS IS SECONDS, AND TEN THOUSANDTHS OFSECONDS-
                        H IS DECLINATION

            EXAMPLE :   LAT-   39-06-51.0701N
                        LONG-  075-27-54.6601W

          B. LAT/LONG VALUES CONVERTED TO ALL SECONDS AS FOLLOWS:

             LATITUDE AND LONGITUDE     SSSSSS.SSSSH
             WHERE :    SSSSSS.SSSS IS THE DEG/MIN/SEC CONVERTED
                                   TO ALL SECONDS
                        H IS THE DECLINATION

      EXAMPLE:  LAT-   140811.0701N
                LONG-  271674.6601W
